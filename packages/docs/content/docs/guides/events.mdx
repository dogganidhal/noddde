## Overview

Events represent facts that have occurred in the domain. In event-sourced aggregates, events are the source of truth — the current state is rebuilt by replaying events through apply handlers.

## DefineEvents

Use `DefineEvents` to declare your event types:

```typescript
import { DefineEvents } from "@noddde/core";

type MyEvents = DefineEvents<{
  SomethingHappened: { data: string };
  SomethingElseHappened: { count: number };
}>;
```
