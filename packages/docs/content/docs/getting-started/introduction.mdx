## What is noddde?

noddde is a TypeScript framework for building applications using Domain-Driven Design (DDD), Command Query Responsibility Segregation (CQRS), and Event Sourcing patterns.

It provides a declarative way to express domain aggregates in terms of state, behaviour (commands and command handlers), and events — following the Decider pattern.

## Core Concepts

- **Aggregates** — Define your domain entities with state, command handlers, and event apply handlers
- **Commands** — Express intent to change state, routed to the appropriate aggregate
- **Events** — Represent facts that have occurred, used to rebuild state and trigger side effects
- **Projections** — Build read models by reacting to events
- **Domain Configuration** — Wire everything together with infrastructure (buses, persistence)
