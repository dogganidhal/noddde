> **ID** = `string` \| `number` \| `bigint`

Defined in: [id.ts:30](https://github.com/dogganidhal/noddde/blob/main/packages/core/src/id.ts#L30)

Union of serializable identifier types supported by the framework. Serves as the upper bound for all aggregate, saga, and entity identifier type parameters. Branded types that extend `string`, `number`, or `bigint` are also accepted.
